# 小破栈

## 前端

vue2、vuex、vue-router、axios、elementui、less、vue-markdown、echarts

## 后端

springboot、mybatisplus

## 其他

MySQL、Git、腾讯云轻量级服务器

## 用户角色

游客（访问网站时的默认角色）

- 浏览网站、包括查看、搜索用户发布、顶、踩的博客、评论（只能看）

普通用户（主动登录账号后）

- 包含游客所有功能
- 额外功能：顶、踩、收藏、发布、编辑、删除博客，顶、踩、发布、删除评论（开放部分功能，可以保存自己的数据）

管理员（主动登录管理员账号）

- 包含游客和普通用户所有功能
- 额外功能：拥有管理后台，可以监控网站使用情况，博客管理、用户管理（开放全部功能）
